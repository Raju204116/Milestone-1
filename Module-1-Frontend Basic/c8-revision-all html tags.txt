1. Document Structure

| Tag               | Purpose                 |
| ----------------- | ----------------------- |
| `<!DOCTYPE html>` | HTML5 declaration       |
| `<html>`          | Root element            |
| `<head>`          | Metadata                |
| `<title>`         | Browser tab title       |
| `<meta>`          | Metadata                |
| `<link>`          | Link CSS, favicon, etc |
| `<style>`         | Internal CSS            |
| `<script>`        | JavaScript              |
| `<body>`          | Visible content         |



2. Text Content

| Tag           | Purpose          |
| ------------- | ---------------- |
| `<h1>`-`<h6>` | Headings         |
| `<p>`         | Paragraph        |
| `<br>`        | Line break       |
| `<hr>`        | Horizontal line  |
| `<strong>`    | Important text/bold  |
| `<b>`         | Bold text        |   
| `<em>`        | Emphasized text/italic  |
| `<i>`         | Italic text      |
| `<mark>`      | Highlighted text |
| `<small>`     | Smaller text     |
| `<sup>`       | Superscript      |
| `<sub>`       | Subscript        |


3. Links & Images

| Tag     | Purpose   |
| ------- | --------- |
| `<a>`   | Hyperlink |
| `<img>` | Image     |


4. Lists

| Tag    | Purpose             |
| ------ | ------------------- |
| `<ul>` | Unordered list      |
| `<ol>` | Ordered list        |
| `<li>` | List item           |
| `<dl>` | Description list    |
| `<dt>` | Description term    |
| `<dd>` | Description details |


6. Containers

| Tag      | Purpose          |
| -------- | ---------------- |
| `<div>`  | Block container  |
| `<span>` | Inline container |

//others semantic tags (class-7)


7. Tables

| Tag       | Purpose      |
| --------- | ------------ |
| `<table>` | Table        |

| `<th>`    | Header cell  |
| `<td>`    | Data cell    |
| `<thead>` | Table head   |
| `<tbody>` | Table body   |
| `<tfoot>` | Table footer |

| Tag          | Purpose         |
| ------------ | --------------- |
| `<form>`     | Form            |
| `<label>`    | Label           |
| `<input>`    | Input field     |
| `<textarea>` | Multi-line text |
| `<button>`   | Button          |
| `<select>`   | Dropdown        |
| `<option>`   | Dropdown option |
| `<fieldset>` | Group fields    |
| `<legend>`   | Fieldset title  |


    
9. Multimedia

| Tag        | Purpose               |
| ---------- | --------------------- |
| `<audio>`  | Audio                 |

| -------------- | ----------------------- |

| `<blockquote>` | Long quotation          |
| `<q>`          | Short quotation         |
| `<details>`    | Expandable content      |
